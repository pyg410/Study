### 특징
- 순서가 보존되는 선형 데이터 구조이다.
- 가장 마지막 요소(가장 최근 삽입된 요소)부터 처리하는 LIFO 이다.(Last In First Out)
- 동적 메모리

### 시간 복잡도

1. **검색 (Search)** : 처음부터 순차적으로 접근해야 하기 때문에 O(n)의 시간 복잡도를 갖는다. 
2. **추가/삭제 (Insert/Delete)** : 가장 위에 데이터를 추가하거나 삭제하기 때문에 O(1)의 시간 복잡도를 갖는다.
